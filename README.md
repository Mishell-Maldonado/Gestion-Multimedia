#  Sistema de Gestión Multimedia - POO

Este proyecto es una aplicación desarrollada en **Java** para la asignatura de Programación Orientada a Objetos. El sistema permite administrar de forma eficiente diversos tipos de contenidos audiovisuales, integrando plataformas tradicionales y modernas.

##  Objetivos y Propósito del Proyecto
El propósito principal es implementar un sistema funcional que gestione información de medios audiovisuales, resolviendo el problema de la fragmentación de datos en plataformas digitales.

**Objetivos académicos:**
- Aplicar los pilares de la POO: **Herencia**, **Encapsulamiento** y **Polimorfismo**.
- Implementar relaciones de **Asociación** y **Agregación** entre objetos.
- Gestionar colecciones de objetos mediante arreglos polimórficos de la superclase.

##  Clases y Funcionalidades Añadidas
Para extender el sistema base, se implementaron las siguientes clases:
- **Clase Twitch:** Gestiona transmisiones en vivo, permitiendo asociar un streamer y un invitado.
- **Clase TikTok:** Maneja contenido de formato corto con atributos específicos como música de fondo.
- **Clases de Soporte (Actor e Investigador):** Permiten la interacción entre el contenido y los colaboradores mediante asociaciones de objetos.
- **ID Automático:** Sistema de identificación único e incremental gestionado desde la superclase.
