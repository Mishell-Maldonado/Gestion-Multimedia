# 🎬 Sistema de Gestión Multimedia - UPS

Proyecto desarrollado para la asignatura de Programación Orientada a Objetos (POO).

## 📝 Descripción del Proyecto
Este sistema permite la gestión y visualización de diversos contenidos audiovisuales como Películas, Series, Documentales, y plataformas modernas como Twitch y TikTok. El objetivo es demostrar el uso de herencia, polimorfismo y relaciones entre clases en Java.

## 🚀 Conceptos Aplicados
- **Herencia:** Superclase abstracta `ContenidoAudiovisual`.
- **Polimorfismo:** Uso de un arreglo único para diferentes tipos de objetos.
- **Relaciones de Asociación:** Conexión de contenidos con las clases `Actor` e `Investigador`.

## ⚙️ Estructura del Proyecto
- `src/`: Carpeta principal con el código fuente organizado por paquetes.
- `PruebaAudioVisual.java`: Clase principal para ejecutar el sistema funcional.
